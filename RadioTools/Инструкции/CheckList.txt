Инструкция пользователя для CheckList.py
Описание
CheckList.py - это инструмент для проверки и очистки плейлистов в формате .pls (формат: track=путь?;). Программа проверяет существование файлов треков и позволяет удалить отсутствующие треки из плейлиста.

Основные возможности
Проверка существования треков локально или на удаленном сервере через SFTP
Автоматическое определение и парсинг формата track=путь?;
Интерактивное удаление отсутствующих треков
Создание резервных копий перед изменением

Использование
python CheckList.py "путь_к_плейлисту.pls"

Шаги работы программы
Шаг 1: Выбор режима проверки
После запуска программа предложит выбрать режим:
1. Локальная проверка - проверка файлов на локальном компьютере
2. Удаленная проверка - проверка файлов на сервере через SFTP

Шаг 2: Настройка подключения (для SFTP)
При выборе удаленной проверки:
Введите хост (по умолчанию: r.dlike.ru)
Введите имя пользователя (по умолчанию: admin)
Введите пароль (ввод скрыт)

Шаг 3: Анализ плейлиста
Программа:
Прочитает и проанализирует файл плейлиста
Извлечет все пути к трекам
Проверит существование каждого файла

Шаг 4: Отчет о проверке
Вы увидите отчет с:
Общим количеством треков
Количеством найденных треков
Количеством отсутствующих треков
Списком отсутствующих треков (если есть)

Шаг 5: Действия с отсутствующими треками
Если найдены отсутствующие треки, программа предложит:
Удалить все отсутствующие треки - автоматическое удаление всех проблемных записей
Выбрать какие треки удалить - ручной выбор конкретных треков для удаления
Не удалять треки - оставить плейлист без изменений
