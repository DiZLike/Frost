Инструкция по использованию скрипта AddToPlaylist.py

Назначение скрипта
Скрипт AddToPlaylist.py предназначен для автоматического создания и обновления плейлиста в формате .pls на основе аудиофайлов в локальной папке. Основные функции:
Поиск новых аудиофайлов в указанной локальной папке
Проверка на дубликаты в существующем плейлисте (нормализация названий треков)
Добавление только новых треков в плейлист
Поддержка различных аудиоформатов

Скрипт поддерживает следующие форматы аудиофайлов:
MP3 (.mp3)
WAV (.wav)
FLAC (.flac)
M4A (.m4a)
AAC (.aac)
OGG (.ogg)
WMA (.wma)
OPUS (.opus)

Подготовка к использованию
1. Установка Python
Убедитесь, что на вашем компьютере установлен Python 3.6 или выше.

2. Структура папок
Серверная папка - путь к файлам на сервере (как они будут отображаться в плейлисте)
Локальная папка - физическое расположение файлов на вашем компьютере
Файл плейлиста - путь к создаваемому/обновляемому плейлисту

3. Нормализация названий
Скрипт автоматически удаляет даты в формате 2025-12-24_12-00_ из названий треков при проверке на дубликаты. Например:
2025-12-24_12-00_MySong.mp3 → mysong
MySong 2025-12-24_12-00_.mp3 → mysong

Использование скрипта
python AddToPlaylist.py <серверная_папка> <локальная_папка> <файл_плейлиста>

Формат плейлиста
Скрипт создает плейлист в формате .pls со следующими записями:
track=/path/to/track1.mp3?;
track=/path/to/track2.mp3?;
