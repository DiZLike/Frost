ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ GAINER.EXE

1. ПОДГОТОВКА:
   - Скопируйте все файлы BASS (.dll) в папку с программой
   - Основные требуемые файлы:
     * bass.dll (обязательно)
     * bassopus.dll (для Opus файлов)
     * bass_aac.dll (для AAC/M4A файлов)
     * bassflac.dll (для FLAC файлов)

2. ИСПОЛЬЗОВАНИЕ:
   gainer.exe -gain n/k -tag n/c -target -23 "путь_к_папке"

   Параметры:
   -gain n/k  : n - без K-фильтра, k - с K-фильтром
   -tag n/c   : n - стандартный тег ReplayGain, c - кастомный тег в комментарии
   -target    : целевое значение LUFS (по умолчанию -23)
   "путь"     : путь к папке с аудиофайлами

   Примеры:
   gainer.exe -gain k -tag c -target -23 "C:\Музыка"
   gainer.exe -gain n -tag n -target -18 "D:\Audio"

3. ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ:
   - MP3, FLAC, WAV, Opus, AAC, M4A, OGG, WMA, MP4, APE, WV

4. ОСОБЕННОСТИ:
   - Программа обрабатывает все файлы рекурсивно (включая подпапки)
   - Отображает прогресс обработки каждого файла
   - Сохраняет рассчитанный Replay Gain в метаданные файла
   - Работает только под Windows
