Доступные шаблонные переменные в паттерне:
{artist} - исполнитель
{title} - название трека
{album} - альбом
{year} - год выпуска
{track} - номер трека (с ведущим нулем)
{trackcount} - общее количество треков в альбоме
{disc} - номер диска
{disccount} - общее количество дисков
{genre} - жанр
{performer} - исполнитель (первый из массива)
{composer} - композитор
{filename} - оригинальное имя файла (без расширения)
{extension} - оригинальное расширение файла
{directory} - относительный путь к файлу (заменит разделители на "-")
